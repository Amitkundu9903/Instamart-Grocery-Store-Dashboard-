🚀 Excited to Showcase My Latest Power BI Project — Instamart Grocery Store Dashboard 📊🛒
Data is powerful when it tells a story — and this project helped me transform raw grocery store data into meaningful business insights using Power BI.
The dashboard is designed to provide a complete overview of sales performance, customer behavior, product trends, and operational KPIs through interactive and visually engaging reports.

🔍 Dashboard Insights Include:
 ✅ Sales & Revenue Performance Tracking
 ✅ Category-wise and Product-wise Analysis
 ✅ Top Performing Products & Cities
 ✅ Customer Purchase Behavior Analysis
 ✅ Profit & Order Trend Monitoring
 ✅ Interactive Filters & Dynamic Visualizations
 ✅ KPI Dashboard for Smart Decision-Making
 
💡 Tools & Skills Used:
 📌 Power BI
 📌 Power Query
 📌 DAX
 📌 Data Cleaning & Transformation
 📌 Data Modeling
 📌 Business Intelligence & Data Visualization
